CodeLive remembers the last document you were working on. Don't worry if you forget the address. I'll redirect you to the exact document when you visit CodeLive again!! @v@)

CodeLive clears every documents left carelessly for a certain period. If you want to keep you works permanently, you have to log in to claim ownership of your documents. If you are logged in, every documents you create are marked as yours, thus they will never disappears.

Features



All common motions and operators, including text objects
Operator motion orthogonality
Visual mode - characterwise, linewise, blockwise
Full macro support (q, @)
Incremental highlighted search (/, ?, #, *, g#, g*)
Search/replace with confirm (:substitute, :%s)
Search history
Jump lists (Ctrl-o, Ctrl-i)
Key/command mapping with API (:map, :nmap, :vmap)
Sort (:sort)
Marks (`, ')
:global
Insert mode
Cross-buffer yank/paste

Because the internal API is quite different from Emacs, there are only a loose approximation of actual Emacs bindings.

Also note that a lot of browsers disallow certain keys from being captured.
For example, Chrome blocks both Ctrl-W and Ctrl-N, with the result that idiomatic use of Emacs keys will close your tab or open a new window.
